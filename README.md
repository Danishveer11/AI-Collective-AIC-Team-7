# AI-Powered Stock Movement Predictor

## Overview

The AI-Powered Stock Movement Predictor is a machine learning web application that predicts whether a stock's price is likely to move **UP** or **DOWN** based on historical stock market data.

This project was developed during a 6-week AI/ML sprint through the AI Collective at UC Davis. Our team designed, developed, and deployed an end-to-end machine learning application that combines data preprocessing, model training, evaluation, and frontend integration.

---

## Problem Statement

Analyzing stock market data can be time-consuming and overwhelming for both beginner and experienced traders. Large volumes of historical market data make it difficult to identify meaningful trends efficiently.

Our project investigates whether machine learning models can learn patterns from historical stock data and provide a simple prediction of future stock movement direction.

---

## MVP Features

* Historical stock data preprocessing and cleaning
* Machine learning prediction pipeline
* Logistic Regression classification model
* XGBoost Classification model
* Random Forest classification model
* Prediction confidence score
* Interactive frontend interface

---

## Demo Workflow

1. User enters stock information.
2. Frontend sends data to the backend API.
3. Backend loads the trained ML model.
4. Model predicts stock movement direction.
5. Frontend displays:

   * Prediction (UP/DOWN)
   * Confidence Score

---

## Machine Learning Approach

### Dataset

The model was trained using historical stock market data containing:

* Open Price
* High Price
* Low Price
* Close Price
* Volume (OHLCV)

### Models Evaluated

During development, we evaluated multiple machine learning models:

1. Logistic Regression (Baseline)
2. Random Forest
3. XGBoost

After comparing model performance, we selected **XGBoost** as our final model due to its strong predictive performance and ability to capture nonlinear relationships within stock market data.

### Evaluation Metrics

Models were evaluated using:

* Accuracy
* Confusion Matrix
* ROC-AUC Score
* Classification Report
